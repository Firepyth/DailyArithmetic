# Daily Arithmetic

A basic web-app daily math game. The goal of the game is to solve as many problems as possible before the time runs out! Every problem solved correctly gives the user a little more time, though with diminishing returns.

# Local Setup

To set up Daily Arithmetic locally, run the following commands:
```
git clone https://github.com/Firepyth/DailyArithmetic.git

cd DailyArithmetic

npm install

npm run dev
```
